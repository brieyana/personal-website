Pet Finder

A web-based application that connects prospective pet owners to local animal shelters. It is loosely inspired by applications that use a “swiping” mechanism for matching.

    • Tested integrity of API with Postman

Beyonce Lyrics Bot

A Twitter bot that posts lyrics from a randomly chosen Beyonce song. Currently the bots only functionality is updating a status.

  • Used Musixmatch’s API to retrieve artist, album, and track data
  • Used Twitter’s API to manage Twitter bot actions, such as posting
  • Parsed lyric data into two line stanzas
  • Deployed application on Heroku for timed execution
